SEC-yield attribution — T-bill + rate carry + credit + structure + floater − ER + residual –
Tip: click any fund's bar to see how its decomposition evolved over time (monthly since 2021).
Residual ≠ manager alpha. It is SEC yield minus (baseline + rate carry + credit + structure + floater − ER). Positive residual can be sec-lending income, tight-spread edge, or security selection — or OAS assumptions being too low (stale hardcoded fallback, FRED currently unreachable). Negative residual can be fee drag beyond ER, tax-equivalence distortion (muni funds SUB/SMMU show negative because baseline is taxable bill yield), or over-pay for risk. Don't attribute this to manager skill without a side-by-side factor regression.
